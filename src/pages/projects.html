recent projects

  • MOONSWELL




    Moonswell is a React SPA (single-page application) made as part of a collaborative hackathon which allows users to seamlessly input a location and generate relevant live data for weather, tidal, and lunar experiences in the specified geographical area using API calls.

    Moonswell was a project my team and I made in part of a collaborative Hackathon event at CoderAcademy. The goal was to create a site in one week using React and Context which we successfully accomplished and styled using Tailwind.

  • FITNESSFINDER API




    FitnessFinder API is a web server API which intends to mitigate and ease the process of finding a new fitness center by allowing owners to log their fitness centre details in a database in order to return relevant information to a user at a specified endpoint.

    I created this API using PostgreSQL as my relational database and normalized it to 3NF. Owners have the ability thru various secure endpoints (JWT auth and verification) to log their facility details. General users can access this information thru separate endpoints to retrieve. Data is output in JSON and can be formatted as required.

  • VIPER TERMINAL APP




    Viper is a language learning terminal application which allows users to learn a language based on pre-populated CSV files.

    With a passion for language, I wanted to create an app that helps students learn their language they're studying. This app features a continuous internal menu meaning all features can be accessed from each other. There's a timed quiz option to utilize when you feel ready, practice section, and a general translation section complemented by the googletrans API (supports all languages).

  • SCRIBBLE




    Scribble is a notetaking app built using React. It persists data and stores your saved notes into local storage.

    Built for fun after learning about hooks, Context and localStorage, I wanted a way to quickly jot down notes without having to use multiple devices. Scribble effectively allows users to type a note, save and edit it as they wish. Close the browser, and your note remains!
